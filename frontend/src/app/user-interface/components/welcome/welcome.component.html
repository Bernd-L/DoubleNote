Welcome to DoubleNote!

To navigate around the application, use the SideNav (a drawer menu which slides in from the left side of the screen) and the CrumbTrail (an interactive indicator of the current location in the application located directly below the blue title bar).

This project is still in its infancy, and not yet considered usable.
Any data stored may become inaccessible due to changes in the software at any point in time for any reason.

Notebook types

This software is supposed to be used for taking notes. There are two different types of notebooks, one called "Box canvas page" and the other called "Sequential block page".

The type of notebook is determined by the user when its created and cannot be changed afterwards.

Box canvas page

The box canvas page (also referred to as BCP) model consists of pages which are not limitend in the way that they are layed out. Users may write down formatted text anywhere on the page by creating a box. The term "canvas" refers to the ability to create free-hand drawings on every such page.

Sequential block page

The sequential block page (also referred to as SBP) model is based on the idea that every page is divided into a linear sequence of blocks, which can be of different types, such as headings, text, picture or other kinds of embeds.

Collaboration

DoubleNote supports two modes of collaboration: Synchronous (real-time) and asynchronous (independant) collaboration.
