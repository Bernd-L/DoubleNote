Notebook import

While DoubleNote-servers are a great choice for backing up your data, syncing it access devices and sharing it with the people you want to, they are not always the available, which is why DoubleNote offers the ability to manually export and import your notebooks.

You can import notebooks from several of sources, such as your clipboard (using JSON) or using DoubleNote's built-in native migration tools.

Transfering notebooks via JSON is less secure, but easier to use, and can be done across multiple devices and browsers, and even between multiple users.

Sharing data using DoubleNote's native migration tools is safer and easier, especially when dealing with multiple notebooks from several sources, but doesn't work across devices for for several users.

Only import notebooks from sources you trust.

Native import

DoubleNote supports migrating (importing and exporting) notebooks from one DoubleNote instance to another. This is accomplished by following these steps:

  1. Open this page in the DoubleNote instance into which you want to import
  2. Copy the import-URL displayed below (you will need it soon)
  3. Open the DoubleNote instance from which you want to export a notebook
  4. On the notebook you want to export, click "export", then click "native export"
  5. Close this page and follow the remaining steps on the other instance.
